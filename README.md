# Hola, soy Javier Baez 👋

Construyo sistemas de automatización con IA para negocios hispanos.

## Qué hago

Diseño y opero asistentes de IA que atienden clientes por WhatsApp,
califican prospectos y dan seguimiento automático — sin perder el trato humano.

- 🤖 Agentes conversacionales de ventas en WhatsApp
- 🔄 Automatización de procesos con n8n
- 📚 Sistemas RAG sobre bases de conocimiento propias
- 📊 Integración de datos, CRM y seguimiento
